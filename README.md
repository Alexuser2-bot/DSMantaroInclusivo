MantaroInclusivos
Descripción Mantaro Inclusivo es una plataforma de turismo accesible diseñada para personas con discapacidades motrices, visuales, auditivas y cognitivas. El proyecto ofrece información detallada sobre destinos turísticos en el Valle del Mantaro (Huancayo, Perú), incluyendo características de accesibilidad, rutas adaptadas y servicios inclusivos.

Objetivos Promover el turismo inclusivo en la región Junín

Proporcionar información detallada sobre accesibilidad de destinos

Facilitar la planificación de viajes para personas con discapacidad

Fomentar el turismo responsable y sostenible

Características Gestión de Usuarios Registro con verificación por email, SMS o 2FA

Inicio de sesión seguro

Perfiles de accesibilidad personalizados (Motriz, Visual, Auditiva, Cognitiva)

Actualización de perfil

Mapa Interactivo Geolocalización del usuario

Marcadores de destinos accesibles

Rutas adaptadas desde la ubicación del usuario

Información de accesibilidad en tiempo real

Catálogo de Destinos Filtrado por nivel de accesibilidad (A, B, C)

Búsqueda por palabra clave

Detalles de características de accesibilidad

Precios y horarios de atención

Asistente IA Recomendaciones personalizadas según perfil

Sugerencias de destinos adaptados

Información de rutas accesibles

Base de Datos Stored Procedures para operaciones CRUD

Índices optimizados para búsquedas

Integridad referencial

Arquitectura El proyecto sigue Arquitectura Hexagonal (Puertos y Adaptadores), garantizando:

Independencia del dominio: Las reglas de negocio no dependen de frameworks externos

Testabilidad: Fácil de probar cada capa de forma aislada

Mantenibilidad: Código organizado y desacoplado

Escalabilidad: Fácil de agregar nuevas funcionalidades
